EU Sales Analysis

screenshot coming soon — drop images/eu-sales.png to populate

This end-to-end interactive sales analysis is built on a live Google Sheets data pipeline, with all metrics calculated dynamically at render time. It features interactive Plotly charts, a diverging year-over-year choropleth, and key takeaways automatically generated from the data and verified for accuracy. The final product is deployed as a self-contained slide deck on GitHub Pages.

The most technically demanding element is the Market Overview slide: a Leaflet choropleth built on rnaturalearth spatial polygons, requiring a geospatial join on country sovereignty fields rather than country names to correctly handle territorial edge cases. The slide combines a live KPI strip, a top-5 countries table, and the map, all assembled in R and rendered as a single output.

This type of interactive sales analysis is a standard business deliverable, commonly produced in Python or BI tools like Tableau. Built in R as part of JHU's [Developing Data Products course](https://www.coursera.org/learn/data-products), it demonstrates the same end-to-end workflow: live data pipeline, interactive visualization, and a single deployable data product.
